Diabetes Prediction using Machine Learning
Project Overview
This project predicts whether a patient is diabetic using machine learning algorithms.
Several classification models were trained and evaluated to determine the best-performing model. According to the project report, Gradient Boosting achieved the best overall performance for deployment.

Dataset

-The dataset contains medical information including:
-Pregnancies
-Glucose
-Blood Pressure
-Skin Thickness
-Insulin
-BMI
-Diabetes Pedigree Function
-Age
-Outcome

Dataset: diabetes.csv

Machine Learning Algorithms

This project evaluates multiple models, including:
-Logistic Regression
-Decision Tree
-Random Forest
-Support Vector Machine (SVM)
-K-Nearest Neighbors (KNN)
-Gradient Boosting
-XGBoost
These algorithms are listed in the accompanying project report.

Best Model: 🏆 Gradient Boosting

Performance:
Accuracy ≈ 91%
The report concludes that Gradient Boosting was selected as the final deployment model due to its strong generalization and performance.
